Homework #2: Weather Widget

Paris

It is 81 degrees outside.

Get Current Temperature

Can you modify weather.js to display the current weather conditions outside?

Rubric:

  1. 15 points: Show Chicago's weather in the widget when the Get Current Temperature button is clicked.

    Finish the code that's been started for you in weather.js to update the widget for Chicago. Chicago's latitude and longitude has been hardcoded for you.

  2. 5 points: Emit the user's actual location into the console.

    Use the browser's navigator.geolocation object to obtain the user's actual real-time latitude and longitude. Don't just show the entire data object provided by the browser. Clearly indicate the latitude and longitude in the console.

  3. 3 points: Use the user's actual location to update the widget.

    The temperature and city name should reflect the user's actual location and weather.
  4. 2 points: Update the weather icon in the widget.
